# 🌐 Websites Projects

A collection of frontend web projects built with HTML, CSS, and SCSS. Each project showcases different web design techniques and practices I've learnt.

## 🛠️ Technologies

- **HTML5**
- **CSS3**
- **Sass/SCSS**
- **Responsive Design**

## 📁 Projects

### 🎨 **App**
A modern website for a portfolio and creative agency. It includes a gallery, blog section, contact forms, and a responsive layout built with CSS Grid.

### 💇 **HairDay**
A boutique hair salon website built with semantic HTML and responsive CSS. It presents services, appointment booking, opening hours, and promotions, with a layout specifically designed to demonstrate cross-platform compatibility across mobile, tablet, and desktop devices.

### 🎬 **Kinomonster**
A movie website for browsing films and TV series. It includes search, ratings, login, and news in a clear and easy-to-use layout.
